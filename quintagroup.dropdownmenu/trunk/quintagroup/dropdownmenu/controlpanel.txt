DropDown Menu Settings
----------------------

All dropdown menu settings reside inside portal_properties tool under
dropdownmenu_properties property sheet. But to adjust it you don't need to
visit ZMI since dropdownmenu package provides control panel for this.

Here are described different dropdown menu related settings and explained how
to set it up.
