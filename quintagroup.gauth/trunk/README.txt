Introduction
============

quintagroup.gauth package suitable for a plone site, which provides access to protected resources of a single google account for a users of the site.

For mentioned task, package provides:
 * configlet for manage of the google account email/password pear.
 * utility, which allow get *email* and/or *password* for access resources of the google account.
 * SafeQuery class, which should be used in successor classes for automatic relogin, if Token is expired.
