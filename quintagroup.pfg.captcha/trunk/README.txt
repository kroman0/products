Introduction
============

quintagroup.pfg.captcha is a package that allows to add captcha field to the form, 
created with PloneFormGen product.

Supported Plone version
-----------------------

* 3.x

Dependencies
------------

* PloneFormGen
* quintagroup.captcha.core

Installation
------------

Please make sure that the following products are intalled in the following order:

* PloneFormGen
* quintagroup.captcha.core
* quintagroup.pfg.captcha

Links
-----
    
* Documentation - http://projects.quintagroup.com/products/wiki/quintagroup.pfg.captcha
* SVN Repository - http://svn.quintagroup.com/products/quintagroup.pfg.captcha

Notes
-----

* If you want to change captcha look - use quintagroup.captcha.core settings: 
http://projects.quintagroup.com/products/wiki/quintagroup.captcha.core

* If you want captcha for Plone default forms - use quintagroup.plonecaptchas product 
http://projects.quintagroup.com/products/wiki/quintagroup.plonecaptchas
