Introduction
============

QGSkel provides skeletons for Plone buildout. 

All skeletons are available as PasteScript_ templates and can be used
via the ''paster'' commandline tool. For example to create a Plone 3 
buildout you can do::

    paster create -t qgplone3_buildout
    
Available templates
===================

Hosting / deployment templates
------------------------------
qgplone3_buildout
  A basic buildout_ based instance for Plone 3.x projects, used in Quintagroup.
