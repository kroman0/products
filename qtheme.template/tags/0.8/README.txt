Introduction
============

ZopeSkel based Theme template with nested namespace.
Support 'addcommad' local command for extend Theme.

This template use ZopeSkel' plone3_theme, and useful
for extend generated theme product with portlet, view
local templates (aka ZopeSkel' archetype template)


Implemented subtemplates for using with 'addcontent'
local command:
    css_resource:    A Plone 3 CSS resource template
    js_resource:     A Plone 3 JS resource template
    skin_layer:      A Plone 3 Skin Layer
    viewlet_hidden:  A Plone 3 Hidden Viewlet template
    viewlet_order:   A Plone 3 Order Viewlet template

Subtemplates use values, entered on base skin generation.
