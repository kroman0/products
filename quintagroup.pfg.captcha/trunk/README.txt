Introduction
============

quintagroup.pfg.captcha is a package that allows to add captcha field to the form, created with PloneFormGen product.

Supported Plone version
-----------------------

* 3.x

Dependencies
------------

* PloneFormGen
* quintagroup.plonecaptchas
* quintagroup.captcha.core

Installation
------------

Please make sure that the following products are intalled in the following order:

* PloneFormGen
* quintagroup.plonecaptchas + quintagroup.captcha.core
* quintagroup.pfg.captcha
