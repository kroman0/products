qPloneEditorRole

This product adds new role named "Editor" and create 'Editors' group.
Also customizes folder workflow, and rolemap.

Editors can only edit content and should not be able to manage Portal
properties.

Compatibility:
 * Plone-2.5.x
 * Plone-2.1.x
