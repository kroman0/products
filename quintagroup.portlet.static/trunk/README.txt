quintagroup.portlet.static
=========================

quintagroup.portlet.static is a Plone product that allows to add Static Stylish Text portlets ot your Plone site. Static Stylish Text portlets are usual Plone Static Text portlets which can have certain portlet style applied. You can choose a css style for the porlet from the list of available css classes.

Supported Plone versions:

    * 3.x
