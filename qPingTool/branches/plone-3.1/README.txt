qPingTool is a simple tool to enable pinging of external feed agregators
for Plone 2.1, Plone 2.5

DEPENDS on:

   * ATContentTypes
   * XMLRPCMethod
   * qRSS2Syndication - to enable RSS2 feeds


INSTALLATION:

1. install it with quickinstaller in Plone.


NOTES:

For sample usage look for qSimpleBlog-pack at http://quintagroup.com/services/plone-development/products/plone-blog
